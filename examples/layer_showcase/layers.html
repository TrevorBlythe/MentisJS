Convolutional layers

These layers are good at analysing images and extracting features from images. They work on any 2D image. You input a one dimensional array formated like this: [r,r,r,r,r,g,g,g,g,g,b,b,b,b,b], (r representing red pixel values, blue representing blue pixel values, green representing green pixel values) (pixel values starting at top left and going to bottom right. This Layer extracts red vertical lines
