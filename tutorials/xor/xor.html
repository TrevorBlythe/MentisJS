XOR Gate tutorial


This is basically the "hello world" of making neural networks. Its very easy and a good benchmark


Getting Started


First understand what the XOR gate is. What we are trying to do it make a network with two inputs and one output. The output will only be on if only one input is on. With that in mind lets make a network that can handle that task. We will use a couple hidden layers (huge overkill) just for funsies.




let net = new Net([
 new FC(1,3),
 new FC(3,3),
 new Sig(3,3),
 new FC(3, 1),
 new Sig(1),
]);

Training it


Now that we have a network. We just train it on the data for awhile and it will learn.




for(var i = 0;i<1000;i++){
net.train([0,0],[0]);
net.train([0,1],[1]);
net.train([1,0],[1]);
net.train([1,1],[0]);
}

Thats it!


Your done with the hard part! Now your network is trained and you can use it whenever you want!




net.forward([0,0]);//outputs ~0
net.forward([0,1]);//outputs ~1
net.forward([1,0]);//outputs ~1
net.forward([1,1]);//outputs ~0
